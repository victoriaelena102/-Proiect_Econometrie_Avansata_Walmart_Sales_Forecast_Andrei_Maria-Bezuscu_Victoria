#Obiectivul proiectului

Scopul principal al cercetării este identificarea factorilor economici, sezonieri și promoționali care influențează nivelul vânzărilor și construirea unor modele predictive capabile să estimeze cât mai precis evoluția acestora.

#Dataset utilizat

Proiectul utilizează setul de date „Walmart Sales Forecast”, care conține informații despre:
- vânzările săptămânale ale magazinelor Walmart;
- departamente comerciale;
- perioade de sărbători;
- variabile economice;
- factori promoționali și sezonieri.

Datele au fost prelucrate și analizate utilizând Python și biblioteci specializate pentru analiză statistică și machine learning.


În cadrul proiectului au fost realizate următoarele etape:
- analiza exploratorie a datelor (EDA);
- tratarea valorilor lipsă;
- analiza corelațiilor dintre variabile;
- feature selection;
- standardizarea datelor;
- evaluarea și compararea mai multor modele predictive;
- analiza erorilor și interpretabilitatea modelelor.

#Modele utilizate

Pentru estimarea vânzărilor au fost utilizate următoarele modele:
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- ARIMA (Time Series Forecasting)

Rezultatele obținute au evidențiat faptul că modelul Random Forest a avut cea mai bună performanță predictivă, obținând:
- R² ≈ 0.967;
- cele mai reduse valori pentru MAE și RMSE;
- o capacitate ridicată de generalizare asupra datelor de testare.

Analiza a demonstrat faptul că sezonalitatea, promoțiile comerciale și diferențele dintre magazine și departamente influențează semnificativ nivelul vânzărilor.
